### Bitcan payment gateway ([Bitcan](https://bitcan.pl/))

## Installation: 

```  
 composer require magespices/bitcan
``` 

and run:
```
    php bin/magento setup:upgrade 
```

to enable modules 

*Warning*: Tested only on Magento 2.3.x Open Source
